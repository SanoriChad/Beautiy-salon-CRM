# Velvet Beauty CRM

Одностраничный дашборд CRM-системы для салона красоты. Создан как портфолио-проект, демонстрирующий навыки UI-разработки.

## Стек

| Технология | Назначение |
|-----------|-----------|
| Vanilla HTML/CSS/JS | без фреймворков |
| [Chart.js 4.x](https://www.chartjs.org/) | графики (CDN) |
| Google Fonts — Inter | типографика |
| Inline SVG (Lucide-style) | иконки |

## Разделы

| Раздел | Описание |
|--------|---------|
| **Дашборд** | 4 метрики, график выручки, популярность услуг, список ближайших записей |
| **Записи** | Расписание по мастерам на день с цветовой разметкой статусов |
| **Клиенты** | Таблица с живым поиском, визитами, статусом VIP |
| **Мастера** | Заглушка (в разработке) |
| **Аналитика** | Заглушка (в разработке) |
