# Velvet Beauty CRM — Portfolio Demo

Одностраничный дашборд CRM-системы для салона красоты. Создан как портфолио-проект, демонстрирующий навыки UI-разработки.

## Стек

| Технология | Назначение |
|-----------|-----------|
| Vanilla HTML/CSS/JS | без фреймворков |
| [Chart.js 4.x](https://www.chartjs.org/) | графики (CDN) |
| Google Fonts — Inter | типографика |
| Inline SVG (Lucide-style) | иконки |

## Разделы

| Раздел | Описание |
|--------|---------|
| **Дашборд** | 4 метрики, график выручки, популярность услуг, список ближайших записей |
| **Записи** | Расписание по мастерам на день с цветовой разметкой статусов |
| **Клиенты** | Таблица с живым поиском, визитами, статусом VIP |
| **Мастера** | Заглушка (в разработке) |
| **Аналитика** | Заглушка (в разработке) |

## Как запустить

```bash

open index.html
# или двойной клик на файле
```

## Особенности реализации

- **Навигация без роутера**: JS переключает `.section.active`, секции скрыты через `display:none`
- **Живой поиск клиентов**: `input` событие + `.filter()` по данным, без debounce (мок-данные)
- **Графики**: Chart.js с кастомным градиентом и тёмным tooltip в стиле бренда
- **Анимации**: `@keyframes fadeUp` при появлении раздела; `transform:translateY(-2px)` на hover карточек
- **Адаптивность**: CSS Grid перестраивается на 1280/1024/768px breakpoints
- **Нет бэкенда**: все данные захардкожены в JS-константах

## Визуальный стиль

- Фон: `#F4F2FF` (лёгкий фиолетовый оттенок)
- Сайдбар: `#1E1B4B` (тёмный индиго)
- Акцент: `#7C3AED` (фиолетовый)
- Карточки: белые, `border-radius: 16px`, `box-shadow` на hover
- Типографика: Inter, иерархия 11/12/13/14/15/21px
